4.标识符

xxxxxxxxxx
C 语言中的标识符是用来命名变量、函数、结构体等程序实体的名称。下面是 C 语言中标识符的命名规则：


1. 标识符必须以字母（大小写均可）或下划线（_）开头。
2. 标识符可以包含字母、数字和下划线。
3. 标识符区分大小写，例如 `count` 和 `Count` 是不同的标识符。
4. 标识符的长度没有限制，但通常建议不要超过 31 个字符。
5. C 语言的关键字不能作为标识符，例如 `int`、`for`、`if` 等。
6. 标识符不能包含空格、标点符号或特殊字符，如 `@`、`#`、`$` 等。
7. 为了提高代码的可读性，应该选择具有描述性的标识符。

5.关键字

xxxxxxxxxx
C 语言中的关键字可以分为以下几类：


1. 数据类型关键字（5 个）：
        char ：用于声明字符类型的变量或函数返回值
        int ：用于声明整型变量或函数返回值
        float ：用于声明单精度浮点型变量或函数返回值
        double ：用于声明双精度浮点型变量或函数返回值
        void ：用于声明无返回值的函数或指针类型
    
2. 存储类关键字（4 个）：
        auto ：定义自动变量，在函数内部声明的变量会被隐式地指定为 `auto`
        register ：用于声明寄存器变量，建议但不保证变量存储在寄存器中
        static ：用于声明静态变量或函数，其生命周期在整个程序执行期间都存在
        extern ：用于声明外部变量或函数，表示它们可以在其他文件中定义


3. 控制流程关键字（6 个）：
        if ：于在条件为真时执行一段代码，否则执行另一段代码
        else ：于在条件语句中定义一个可选的执行分支，当条件不满足时执行
        switch ：于实现多分支的条件语句，根据表达式的值执行相应的分支
        case ：switch 语句中用于定义各个分支的匹配值
        default ：在 `switch` 语句中用于定义默认分支
        while ：用于创建一个循环，只要指定的条件为真，就一直执行一系列语句


4. 循环关键字（3 个）：
        for ：用于创建一个循环，在指定条件为真时循环执行一系列语句
        do ：用于创建一个循环，至少执行一次循环体，然后根据条件决定是否继续执行
        while ：用于创建一个循环，只要指定的条件为真，就一直执行一系列语句


5. 跳转关键字（3 个）：
        break ：用于跳出循环或 `switch` 语句，并继续执行程序的下一行代码
        continue ：用于结束当前循环的迭代并继续进行下一次迭代
        goto ：用于无条件地将控制转移到程序中的标签位置


6. 其他关键字（11 个）：
        const ：用于定义常量，表示变量的值不能被修改
        sizeof ：用于获取类型或变量的大小（以字节为单位）
        enum ：用于定义枚举类型。枚举是一组命名的整数常量
        typedef ：用于给现有类型定义新的名称，可以提高代码的可读性和灵活性
        struct ：用于定义结构体类型，可以封装多个不同类型的变量
        union ：用于定义联合类型，多个成员共享同一块内存空间
        volatile ：用于声明易变的变量，表示变量的值可能会在意料之外的情况下发生变化
        signed ：用于声明有符号整型变量
        unsigned ：用于声明无符号整型变量
        long ：用于声明长整型变量或函数返回值
        short ：用于声明短整型变量或函数返回值

                 switch (expression) {
                     case constant1:

                //2.数组在使用前必须确定开辟内存的大小，使用起来极为不便，C语言的链表更灵活，C++的容器更好用


1. 数组的定义和初始化
           数组的定义使用方括号`[]`表示数组的长度，并且需要指定数组元素的类型。数组的初始化可以在定义时进行，也可以在后续的代码中逐个赋值


           // 定义一个包含5个整数的数组并初始化
           int numbers[5] = {1, 2, 3, 4, 5};




2. 数组的访问和操作
           数组的元素可以通过索引（下标）来访问和操作，数组的索引从0开始，到数组长度减1结束。可以使用赋值语句对数组元素进行赋值或修改


           int x = numbers[0]; // 访问数组的第一个元素
           numbers[1] = 10;    // 修改数组的第二个元素




3. 多维数组
           多维数组是指数组中包含其他数组作为元素的数组。多维数组可以理解为表格或矩阵，可以通过多重索引来访问和操作元素


           // 定义一个2行3列的二维数组并初始化
           int matrix[2][3] = {{1, 2, 3}, {4, 5, 6}};
           int x = matrix[0][1]; // 访问第一行第二列的元素
        
            //多维数组中，每一层各个子元素维数必须相同（二维是长方形，三维是长方体）




4. 数组的长度和sizeof运算符
           数组的长度可以通过`sizeof`运算符来获取，它返回数组所占用的字节数。可以使用`sizeof`来计算数组的长度


           int length = sizeof(numbers) / sizeof(numbers[0]); // 数组的长度为5




5. 数组作为函数参数
           数组可以作为函数的参数传递给函数，可以将整个数组或指向数组的指针作为参数。在函数内部，可以通过指针操作数组的元素


           void printArray(int arr[], int length) {
               for (int i = 0; i < length; i++) {
                   printf("%d ", arr[i]);
               }
               printf("\n");
           }




6. 数组和指针的关系
           数组名本身可以被看作一个指针，它存储了数组的首地址。通过指针算术运算可以遍历数组的元素，也可以使用指针访问和操作数组的元素


           int* ptr = numbers;  // 数组名可以看作指针
           int x = *ptr;        // 指针访问数组的第一个元素
           int y = *(ptr + 1);  // 指针访问数组的第二个元素




7. 字符串和字符数组
           字符数组是特殊的数组形式，用于存储字符串。C语言没有内置的字符串类型，而是使用以空字符（'\0'）结尾的字符数组来表示字符串


           char str[] = "Hello"; // 定义并初始化字符数组




8. 数组的遍历和操作

           for (int i = 0; i < length; i++) {

           }

指针：存储变量地址的变量，通过指针可以直接访问和操作内存中的数据，很强大，但也很危险（随意更改内存中的数据，可能会导致系统崩溃）


1. 指针的定义和初始化
        变量类型 *变量名;      //定义指针变量（变量类型，表示指针变量所存储的地址所保存的变量的类型）

            取地址：使用&可以获取值存在哪个地址中（值--->地址）


           int x = *ptr;    // 解引用指针，获取指针（地址）所指向的值

3. 空指针
            空指针，即没有指向任何内存的指针，其值为NULL，空指针在使用前要赋一个确定的地址，否则很危险
 
           int *ptr = NULL;   // 初始化为空指针
           if (ptr == NULL) { // 判断指针是否为空

           }
 


4. 指针的算术运算

           char str[]="asdfghjkl";

5. 指针和数组的关系
           数组名就是数组第一个元素的地址，可以用指针操作数组


           int arr[] = {1, 2, 3, 4, 5};
           int *ptr = arr;     // 数组名可以看作指针
           int x = *ptr;       // 指针访问数组的第一个元素
           int y = *(ptr + 1); // 指针访问数组的第二个元素

                                        如果要想改变值，可以通过返回值，将外部变量赋值为函数的返回值

7. 动态内存分配
           使用`malloc`函数可以在程序运行时动态地分配内存空间。分配得到的内存可以通过指针进行访问和操作，使用完毕后需要使用`free`函数释放内存


           int *ptr = (int *)malloc(sizeof(int));  // 分配一个整数大小的内存
           *ptr = 10;                             // 对分配得到的内存进行赋值
           free(ptr);                             // 释放内存




8. 指针和字符串
           字符串在C语言中是以字符数组的形式存储的，因此指针可以用于访问和操作字符串。常见的字符串操作函数如`strcpy`、`strlen`等接受指针作为参数


           char str[] = "Hello";
           char *ptr = str;                       // 指针指向字符串的首地址
           int length = strlen(ptr);              // 计算字符串的长度
           char *copy = strcpy((char *)malloc(length), ptr); // 复制字符串




9. 指针的安全性和悬挂指针
           指针的错误使用可能会导致程序崩溃或产生不确定的结果。悬挂指针是指指向已释放的内存区域或未初始化的指针，访问悬挂指针会导致不可预料的结果


           int *ptr = (int *)malloc(sizeof(int));
           free(ptr);
           *ptr = 10;  // 访问已释放的内存，发生错误




10. 指针数组
            指针数组是一个数组，其中的每个元素都是指针类型。可以使用指针数组来存储一组指针，每个指针可以指向不同的数据


            int num1 = 10, num2 = 20, num3 = 30;
            int *ptrArray[] = {&num1, &num2, &num3};   // 定义一个指针数组
            int **ptr = ptrArray;                      // 指针数组的指针

11. 函数指针
            函数指针是指向函数的指针变量，它可以用于间接调用函数。函数指针的声明需要指定函数的返回值类型和参数列表


            int add(int a, int b) {
                return a + b;
            }


            int (*funcPtr)(int, int);   // 函数指针的声明
            funcPtr = add;              // 将函数的地址赋给函数指针
            int result = funcPtr(3, 4); // 通过函数指针调用函数
            printf("%d\n", result);     // 输出函数调用的结果




12. 指向指针的指针
            指向指针的指针表示一个指针变量的地址，可以用于操作二级指针所指向的内存区域

13. const 修饰符

            int num = 10;

C语言中的结构（structure）是一种用户自定义的数据类型，允许将不同类型的数据组合到一个单独的实体中。结构由一系列成员变量组成，每个成员变量可以是不同的数据类型


1. 定义结构
    
        // 定义结构体
        struct 结构体名称 {
            // 成员变量
            数据类型 成员变量1;
            数据类型 成员变量2;
            // ...
        };
    
       // 使用typedef为结构体定义别名（可选）
        typedef struct 结构体名称 别名;




    例子：
                struct Student {
                    char name[20];
                    int age;
                    float score;
                };


                typedef struct struct Student {
                    char name[20];
                    int age;
                    float score;
                }student;




2. 声明结构变量
        定义结构之后，可以声明结构变量来使用它。结构变量的声明方式与普通变量相似，只需在结构名前加上关键字struct


            struct Student stu1; // 声明一个名为stu1的结构变量




3. 访问结构成员
        可以使用成员运算符`.`来访问结构成员。成员运算符连接结构变量名和成员变量名


        strcpy(stu1.name, "Tom"); // 赋值给结构成员name
        stu1.age = 18; // 赋值给结构成员age
        stu1.score = 95.5; // 赋值给结构成员score




4. 结构的初始化
        可以在声明结构变量时对其进行初始化。初始化结构时，按照成员变量在结构中的声明顺序给出初始值


        struct Student stu2 = {"Jack", 20, 88.5}; // 初始化结构变量stu2




5. 结构作为函数参数和返回值
        结构可以作为函数的参数和返回值。在函数参数列表中，可以使用结构类型作为参数类型，并传递结构变量


        void printStudent(struct Student stu) {
            printf("Name: %s\n", stu.name);
            printf("Age: %d\n", stu.age);

        }


        // 调用函数
        printStudent(stu1);




        在函数声明时，可以使用结构类型作为返回值类型


        struct Student createStudent() {
            struct Student stu;
            // 初始化结构变量stu
            return stu;
        }




6. 结构嵌套
        在一个结构中可以包含另一个结构作为成员变量，这被称为结构嵌套。通过使用嵌套结构，可以创建更复杂的数据结构


        struct Date {
            int year;
            int month;
            int day;
        };


        struct Person {
            char name[20];

        };


        // 使用结构嵌套来访问成员变量
        struct Person p;
        p.birthdate.year = 1990;
        p.birthdate.month = 9;
        p.birthdate.day = 10;




7. 结构指针
        可以通过指向结构的指针来操作结构。使用`->`运算符来访问结构指针所指向的成员变量


        struct Student *ptr = &stu1; // 定义一个指向结构的指针
        printf("Name: %s\n", ptr->name);
        printf("Age: %d\n", ptr->age);
        printf("Score: %.1f\n", ptr->score);




8. 结构数组
        可以创建结构数组，其中每个数组元素都是一个结构变量


        struct Student students[3]; // 声明一个包含3个学生的结构数组


        for (int i = 0; i < 3; i++) {

        使用sizeof运算符可以计算结构的大小，即所占用的字节数。结构的大小由结构中各个成员变量的大小决定


        printf("Size of struct Student: %zu bytes\n", sizeof(struct Student));




        10. 结构的作用域
        结构的作用域与其他变量类似，可以在函数内部或外部定义结构。如果要在多个函数之间共享结构，建议将结构定义在全局作用域中。

        11. 结构的对齐
        结构的成员变量在内存中的存储通常是按照特定的对齐方式进行的。这是为了提高访问成员变量的效率。可以使用`#pragma pack(n)`指令来控制结构的对齐方式，其中n表示对齐字节数。

void insertAtEnd(struct Node** head, int data) {

   newNode->data = data;
   newNode->next = NULL;


   if (*head == NULL) {
      *head = newNode;
   } else {
      struct Node* temp = *head;
      while (temp->next != NULL) {
         temp = temp->next;
      }
      temp->next = newNode;
   }
}

void insertAtPosition(struct Node** head, int data, int position) {
   if (position == 0) {  // 在开头插入
      insertAtBeginning(head, data);
   } else {
      struct Node* newNode = (struct Node*)malloc(sizeof(struct Node));
      newNode->data = data;


      struct Node* temp = *head;
      for (int i = 0; i < position - 1 && temp != NULL; i++) {
         temp = temp->next;
      }
      if (temp == NULL) {
         printf("Invalid position\n");
         return;
      }
      newNode->next = temp->next;
      temp->next = newNode;
   }
}

void deleteNode(struct Node** head, int key) {
   struct Node* temp = *head;
   struct Node* prev = NULL;


   if (temp != NULL && temp->data == key) {
      *head = temp->next;
      free(temp);
      return;
   }


   while (temp != NULL && temp->data != key) {
      prev = temp;
      temp = temp->next;
   }


   if (temp == NULL) {
      printf("Key not found\n");
      return;
   }


   prev->next = temp->next;
   free(temp);
}

8.内存管理

xxxxxxxxxx
1.内存空间主要分为以下几个区域：


        1. 栈（Stack）：
            - 作用：栈用于存储局部变量、函数参数以及函数调用时的上下文信息。
            - 特点：栈是一种自动分配和释放内存的机制，它的大小是有限的。
                    栈中的数据可以通过指针进行快速访问，但生命周期受限于函数的执行。


        2. 堆（Heap）：
            - 作用：堆用于动态分配内存，即通过malloc()、calloc()等函数在运行时分配的内存空间。
            - 特点：堆的大小通常比栈大，并且在程序运行期间可以动态增长。
                    堆中的数据可以通过指针进行访问和操作，需要手动释放已分配的内存空间。


        3. 全局区（Global/Static Data Segment）：
            - 作用：全局区用于存储全局变量和静态变量。
             - 特点：全局区在程序启动时分配，在程序结束时释放。
                    全局区中的数据可以在整个程序中被访问和修改。


        4. 常量区（Constant Data Segment）：
            - 作用：常量区用于存储常量数据，如字符串常量。
            - 特点：常量区中的数据不可修改，只能读取。


        5. 代码区（Code Text Segment）：
            - 作用：代码区用于存储程序的机器指令。
             - 特点：代码区中的数据是只读的。


                    这些内存区域按照不同的作用和特点，用于存储不同类型的数据。了解这些区域对于理解内存管理、变量的作用域和生命周期等概念非常重要。






2. 动态内存分配：C语言中提供了动态内存分配的功能，通过malloc()函数来在运行时分配内存空间。需要注意的是，在使用完之后，需要使用free()函数释放已分配的内存空间，以防止内存泄漏。

3. 指针操作：指针是C语言中重要的概念，可以通过指针操作来访问和修改内存中的数据。例如，使用指针访问数组元素：

4. 内存拷贝：C语言中提供了memcpy()函数，用于将一块内存的内容拷贝到另一块内存中。

int main() {

    SOCKET server_sock, client_sock; // 服务端和客户端的socket文件描述符
    struct sockaddr_in server_addr, client_addr; // 服务端和客户端的地址信息
    int addr_len, bytes_received; // 客户端地址信息的长度，接收到的字节数

    if (WSAStartup(MAKEWORD(2, 2), &wsa) != 0) { // MAKEWORD用于构造网络字节序的版本号
        printf("WSAStartup failed. Error Code : %d", WSAGetLastError()); // 打印错误信息并退出程序
        return 1;
    }

    if ((server_sock = socket(AF_INET, SOCK_STREAM, 0)) == INVALID_SOCKET) { // 创建TCP套接字
        printf("Could not create socket : %d", WSAGetLastError()); // 打印错误信息并退出程序
    }


    // 配置socket地址
    server_addr.sin_family = AF_INET; // 使用IPv4地址族
    server_addr.sin_addr.s_addr = INADDR_ANY; // 绑定所有可用的网络接口
    server_addr.sin_port = htons(PORT); // 设置端口号


    // 绑定socket到指定地址和端口
    if (bind(server_sock, (struct sockaddr*)&server_addr, sizeof(server_addr)) == SOCKET_ERROR) { // 将socket绑定到指定地址和端口
        printf("Bind failed with error code : %d", WSAGetLastError()); // 打印错误信息并退出程序
        return 1;
    }


    // 监听连接
    if (listen(server_sock, SOMAXCONN) == SOCKET_ERROR) { // 启动监听，等待客户端连接
        printf("Listen failed with error code : %d", WSAGetLastError()); // 打印错误信息并退出程序
        return 1;
    }


    printf("Server listening on port %d\n", PORT); // 打印服务端监听的端口号


    // 接受客户端连接
    addr_len = sizeof(client_addr); // 获取客户端地址信息的长度
    if ((client_sock = accept(server_sock, (struct sockaddr*)&client_addr, &addr_len)) == INVALID_SOCKET) { // 接受客户端连接并返回一个新的socket文件描述符
        printf("Accept failed with error code : %d", WSAGetLastError()); // 打印错误信息并退出程序
        return 1;
    }


    // 向客户端发送欢迎消息
    const char* welcome_message = "Welcome to the server!"; // 定义欢迎消息
    send(client_sock, welcome_message, strlen(welcome_message), 0); // 发送欢迎消息给客户端


    while (1) {
        // 接收客户端消息
        memset(client_message, 0, sizeof(client_message)); // 清空缓冲区
        bytes_received = recv(client_sock, client_message, sizeof(client_message), 0); // 接收客户端消息


        if (bytes_received > 0) { // 如果接收到了消息
            printf("Client: %s\n", client_message); // 打印客户端发送的消息


            // 处理客户端消息（这里只是简单地回显消息）
            send(client_sock, client_message, bytes_received, 0); // 发送消息给客户端
        }
        else if (bytes_received == 0) { // 如果客户端主动断开连接
            printf("Client disconnected\n"); // 打印客户端已断开连接的消息
            break; // 跳出循环
        }
        else { // 如果接收失败
            printf("Recv failed with error code : %d", WSAGetLastError()); // 打印错误信息并退出程序
            break; // 跳出循环
        }
    }


    // 关闭socket文件描述符，释放资源
    closesocket(client_sock);
    closesocket(server_sock);
    WSACleanup();


    return 0;
}

    if (WSAStringToAddressW((wchar_t*)src, af, NULL, (struct sockaddr*)&ss, &size) == 0) { // 将点分十进制的IP地址转换为网络字节序的二进制格式，并存储到dst指针所指向的内存中

    }

        return -1;
    }

    memset(server_reply, 0, sizeof(server_reply)); // 清空缓冲区

        printf("Server: %s\n", server_reply); // 打印服务端发送的消息
    }

            printf("Server: %s\n", server_reply); // 打印服务端发送的消息
        }
    }

    closesocket(sock);
    WSACleanup();


    return 0;
}
3.区别
xxxxxxxxxx
客户端和服务端的代码在一些地方有所不同，因为它们扮演着不同的角色。


1. 服务端代码：
   - 创建socket文件描述符并进行绑定、监听。
   - 使用`accept()`函数等待客户端连接，并接受连接请求。
   - 发送欢迎信息给客户端。
   - 进入循环，接收来自客户端的消息并回复。


2. 客户端代码：
   - 创建socket文件描述符。
   - 设置服务端的IP地址和端口号。
   - 使用`connect()`函数连接到服务端。
   - 接收服务端发送的欢迎消息。
   - 进入循环，发送消息给服务端并接收回复。


总体上，服务端负责监听和接受客户端连接，处理客户端的请求；而客户端负责连接服务端，发送请求并接收服务器的响应。


需要注意的是，两者都使用了WinSock库来实现网络通信，但服务端代码中包含了额外的绑定、监听、接受连接等步骤，而客户端代码中则包含了连接操作和与服务端的交互。

7.SSL/TLS

SSL（Secure Sockets Layer）和TLS（Transport Layer Security）是用于保护网络通信安全的协议，可以防止数据被窃听、篡改和伪造。

在网络编程中，可以使用SSL/TLS来加密和解密数据，保护通信安全。

        SSL_set_fd(ssl, client_sock);

        if (SSL_accept(ssl) <= 0) {
            ERR_print_errors_fp(stderr);

8.多线程编程

在网络编程中，多线程编程可以提高系统的并发性能，同时也可以避免阻塞和死锁问题。

可以使用pthread库在C语言中进行多线程编程。

// 服务端代码
#include <stdio.h>
#include <stdlib.h>
#include <string.h>
#include <sys/socket.h>
#include <arpa/inet.h>
#include <pthread.h>


void* handle_client(void* arg);


int main() {
    int sockfd, client_sock;
    struct sockaddr_in server_addr, client_addr;


    // 创建Socket
    sockfd = socket(AF_INET, SOCK_STREAM, 0);


    // 设置服务器地址
    server_addr.sin_family = AF_INET;
    server_addr.sin_port = htons(8080);
    server_addr.sin_addr.s_addr = htonl(INADDR_ANY);


    // 绑定Socket到指定地址和端口
    bind(sockfd, (struct sockaddr*)&server_addr, sizeof(server_addr));


    // 监听连接请求
    listen(sockfd, 5);


    // 接受客户端连接
    unsigned int client_len = sizeof(client_addr);
    pthread_t thread_id;
    while ((client_sock = accept(sockfd, (struct sockaddr*)&client_addr, &client_len)) != -1) {
        // 创建线程处理客户端连接
        if (pthread_create(&thread_id, NULL, handle_client, (void*)&client_sock) != 0) {
            perror("pthread_create");
        }
        pthread_detach(thread_id);
    }


    // 关闭Socket
    close(sockfd);


    return 0;
}

9.epoll

epoll是一种高效的I/O多路复用机制，可以大大提高网络编程的并发性能。

在Linux系统中，可以使用epoll库实现epoll机制。

xxxxxxxxxx
// 服务端代码
#include <stdio.h>
#include <stdlib.h>
#include <string.h>
#include <sys/socket.h>
#include <arpa/inet.h>
#include <unistd.h>
#include <sys/epoll.h>


int main() {
    int sockfd, client_sock;
    struct sockaddr_in server_addr, client_addr;


    // 创建Socket
    sockfd = socket(AF_INET, SOCK_STREAM, 0);


    // 设置服务器地址
    server_addr.sin_family = AF_INET;
    server_addr.sin_port = htons(8080);
    server_addr.sin_addr.s_addr = htonl(INADDR_ANY);


    // 绑定Socket到指定地址和端口
    bind(sockfd, (struct sockaddr*)&server_addr, sizeof(server_addr));


    // 监听连接请求
    listen(sockfd, 5);


    // 创建epoll对象
    int epfd = epoll_create(1);
    if (epfd == -1) {
        perror("epoll_create");
        exit(EXIT_FAILURE);
    }


    // 添加监听Socket到epoll对象
    struct epoll_event ev;
    ev.events = EPOLLIN;
    ev.data.fd = sockfd;
    if (epoll_ctl(epfd, EPOLL_CTL_ADD, sockfd, &ev) == -1) {
        perror("epoll_ctl");
        exit(EXIT_FAILURE);
    }


    // 处理事件
    struct epoll_event events[10];
    int nfds;
    char buffer[1024];
    while (1) {
        nfds = epoll_wait(epfd, events, 10, -1);
        if (nfds == -1) {
            perror("epoll_wait");
            break;
        }


        for (int i = 0; i < nfds; i++) {
            if (events[i].data.fd == sockfd) {
                // 接受客户端连接
                unsigned int client_len = sizeof(client_addr);
                client_sock = accept(sockfd, (struct sockaddr*)&client_addr, &client_len);


                // 添加客户端Socket到epoll对象
                ev.events = EPOLLIN | EPOLLET;
                ev.data.fd = client_sock;
                if (epoll_ctl(epfd, EPOLL_CTL_ADD, client_sock, &ev) == -1) {
                    perror("epoll_ctl");
                    exit(EXIT_FAILURE);
                }
            } else {
                // 处理客户端连接
                int client_sock = events[i].data.fd;
                int n = recv(client_sock, buffer, sizeof(buffer), 0);
                if (n == 0) {
                    // 客户端连接关闭
                    close(client_sock);
                    epoll_ctl(epfd, EPOLL_CTL_DEL, client_sock, NULL);
                } else if (n > 0) {
                    // 接收数据并发送数据
                    printf("Received data from client: %s\n", buffer);
                    send(client_sock, "Hello, client!", sizeof("Hello, client!"), 0);
                } else {
                    // 出错
                    perror("recv");
                    close(client_sock);
                    epoll_ctl(epfd, EPOLL_CTL_DEL, client_sock, NULL);
                }
            }
        }
    }


    // 关闭Socket和epoll对象
    close(sockfd);
    close(epfd);


    return 0;
}

#include <string.h>

11.预处理

xxxxxxxxxx
以下是关于C语言预处理的全部知识笔记：


1. 什么是预处理
在C语言中，预处理是在编译之前对源代码进行的一系列文本替换和条件编译操作。预处理器扫描源代码，根据预处理指令对代码进行修改，生成新的代码，并在编译之前将其提供给编译器。


2. 预处理指令
预处理指令以井号（#）开头，用于告诉预处理器执行相应的操作。常见的预处理指令有：


- #include：包含一个头文件的内容。
- #define：定义一个宏。
- #ifdef / #ifndef：条件编译，根据某个宏是否已定义来决定编译的部分。
- #if / #elif / #else / #endif：条件编译，根据给定的条件表达式来决定编译的部分。
- #error：生成一个编译错误。
- #pragma：用于控制编译器的行为。


示例：
#include <stdio.h>


#define MAX_NUM 100


#ifdef DEBUG
    #define LOG(msg) printf("[DEBUG] %s\n", msg)
#else
    #define LOG(msg)
#endif


int main() {
    int i;


    for (i = 0; i < MAX_NUM; i++) {
        LOG("Processing...");
        // 其他操作...
    }


    return 0;
}






3. 宏定义
宏定义是使用预处理指令#define定义的标识符，用于代表一个值或一段代码片段。宏定义使用简单的文本替换来将标识符替换为相应的值或代码。


示例：


#define MAX(a, b) ((a) > (b) ? (a) : (b))


int main() {
    int x = 10;
    int y = 20;
    int max = MAX(x, y);


    printf("最大值是：%d\n", max);


    return 0;
}




4. 文件包含
通过预处理指令#include，可以将一个头文件的内容包含到当前源文件中。这样可以将常用的函数声明、宏定义等放在头文件中，方便代码的复用和维护。


示例：
// utils.h


#ifndef UTILS_H
#define UTILS_H


void printMessage();


#endif




// utils.c


#include "utils.h"
#include <stdio.h>

    printf("Hello, World!\n");
}




// main.c


#include <stdio.h>
#include "utils.h"


int main() {
    printMessage();


    return 0;
}

C语言中有一些预定义的宏，可以在编译过程中使用。常见的预定义宏有：


- __FILE__：当前源文件的文件名。
- __LINE__：当前行号。
- __DATE__：当前的日期。
- __TIME__：当前的时间。
- __cplusplus：如果当前编译器支持C++，则定义为一个非零值。
